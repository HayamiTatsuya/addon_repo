Addon Contains:
- Trench Modules - Consists of modular parts for building trenches, can be found under Empty->Structures->RnT Structures

Features:
Trench Modules consists of 
- Bunker module
- Corner module
- End module
- Straight module
- Stand module
- Position module
- 'T' module
