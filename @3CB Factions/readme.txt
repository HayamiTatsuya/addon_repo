Version 8.0.1: 3CB Factions

Installation
------------
1. Unzip the folder and place in the root of Arma 3 installation. 
2. Add the mod name to the start-up using either a launcher or by adding "-mod=@3cb_factions". 
3. Copy the ".bikey" file to the "Keys" folder in Arma 3 root.
